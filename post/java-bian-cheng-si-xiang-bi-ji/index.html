Java编程思想笔记

1.对象导论

1.1抽象过程

万物皆对象。

程序是对象的集合，通过发送消息来告诉彼此所需要做的。：将消息比作某特定对象的方法的调用请求。

每个对象都有自己的由其它对象构成的存储。：对象中包含另一个对象。

每个对象都有其类型。

某一特性类型的所有对象都能接收同样的消息。 ：多态

1.2每个对象都有一个接口

抽取出具有相同特性（属性）和行为（方法）的对象集合。

比如向某个对象发送消息（发送请求），该对象能够知道该消息的目的并能够执行相应的代码。

1.3每个对象都提供服务

设计程序时，将对象视为服务提供者。程序向用户提供服务，然后程序调用对象的提供的服务以此实现目的。

同时将对象视为服务提供者能够提高对象的高内聚，对象完成提供服务的使命，但是绝不多做任何事。

1.4被隐藏的具体实现

程序中有服务提供者与服务调用者，服务提供者只需要向服务调用者暴露后者所需要的部分即可，减少不必要的bug。例如我们需要将数据库中的完整PO对象转换成业务层中的DTO对象，并且仅提供业务层中需要的属性。

同时还可使用访问控制关键字加以保护，public、protected、private以及default进行控制。

1.5复用具体实现

某一对象作为另一对象的属性 组合（has-a）

两个对象动态进行组合，例如：方法参数。聚合组合的依赖性比聚合要高

1.6继承

is-a继承中子类重写父类中的方法

is-like-a子类新增了父类中不具有的方法

1.7伴随多态的可互换对象

多态即是：父引用指向子实例，实际对应逻辑是子实例中的逻辑

2.一切都是对象

2.1用引用操纵对象

持有对象的引用(地址)，即可对对象进行操作。Java是值传递，引用对象传递的是其地址。

2.2必须由你来创建对象

2.2.1存储到什么地方

2.2.1.1寄存器

位于处理器内部，寄存器内存有限，用户无感且无法控制。

2.2.1.2堆栈

位于RAM中，通过堆栈指针进行控制，若指针向下移动，则是分配新的内存，指针向上移动，则是释放内存。大多的Java对象引用存放于堆栈中

2.2.1.3 堆

通用的内存池，位于RAM中，用于存放所有Java对象，

2.2.1.4常量存储

存放在Rom中。

2.2.1.5非RAM存储

持久化对象，存放在磁盘上。

2.2.2基本类型

基本数据类型是放在栈中还是放在堆中，这取决于基本类型声明的位置。

基本数据类型变量在局部方法中是存放在栈中（变量名及其值）；在实例对象中是存放在堆中（变量名及其值）。

3.容器

3.1迭代器

3.1.1 Iterator

迭代器（一种设计模式）是一个对象，它的工作是遍历并选择序列（集合）中的对象。Java的Iterator只能单向移动。这个Iterator主要用来：

  1. 使用方法**Iterator()**要求容器返回一个Iterator对象。Iterator将准备返回序列（集合）中的第一个元素。
  2. 使用**next()**获得序列（集合）中的下一个元素。
  3. 使用**hasNext()**检查序列（集合）中是否还有元素。
  4. 使用**remove()**将迭代器最新返回的元素进行删除。

3.1.2 ListIterator

ListIterator是Iterator的子类型，只能用于List之间的各种访问，ListIterator可以双向移动，同时可以产生相对迭代器在列表中只想当前位置的前一个和后一个的索引，并且可以使用set()方法替换它访问过的最后一个元素，同时还可以通过调用**listIterator(index)**创建一个一开始就指向索引为index处的ListIterator

public static void main(String[] args) {
        List<String> list = Arrays.asList("s1","s2","s3","s4","s5","s6","s7","s8","s9");
        ListIterator<String> iterator = list.listIterator();
        while (iterator.hasNext()) {
            System.out.println("previousIndex:"+iterator.previousIndex()+"\tnextIndex:"+iterator.nextIndex()+"\tnextValue:"+iterator.next());
        }
		
    	//指向索引为3的迭代器
        iterator = list.listIterator(3);
        while (iterator.hasNext()) {
            iterator.next();
            //同时将对应值进行替换
            iterator.set("zero");
        }
        System.out.println(list);
    }
